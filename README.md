## Instalação

Clone o projeto com

```sh
git clone https://github.com/marcoscosta90/mezaChallenge.git
```

Entre na pasta do projeto e instale as dependências com:

```sh
npm i
```

Após a instalação dos pacotes, você pode iniciar a aplicação com:

```sh
npm start
```
